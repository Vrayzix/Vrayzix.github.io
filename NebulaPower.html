Nebula Power
Platform: PC,Android
Project Type: Personal Project
Software Used: Unity
Language Used: C#
PLAY

Nebula Power is a pixel classic genre game i made using the Unity engine.

The game is about a spaceship traveling across the galaxy defeating enemies and challenging dangerous bosses on it's way.

The game is mainly made for android platforms but i have also added a PC version.

Game Mechanics:

The game progression works with waves and levels system,each level have a particular set of waves,at the end of each wave you will fight a boss to unlock the next level.

Currently there is only one level but more will be added soon...

Your spaceship have the ability to shoot fireballs which damage any enemy in it's trajectory and there are also specific cooldowns in order to shoot again.

User Interface:

For user interface implementation i used a very simple yet also responsive interface in order for the player to focus more on the gameplay and not being overwhelmed and distracted by the UI's.

Moreover, i used a galaxy space theme background with planets and stars in order for the player to get full immersion in the world of the game.

Currently in the menu, there is only a play button but as the game features expand in the future, i will add more options for example: game settings, a shop and so on..
